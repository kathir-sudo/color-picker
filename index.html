Section 3: Advanced Debugging (2 Questions x 10 Marks = 20 Marks)

  1. Objective: Create a function to find the largest number in a list.
    Problem: The function works for positive numbers but fails for lists containing only negative numbers.
    Expected Output for Case 2: Max of [-10, -5, -2] is: -2 
    def find_max(numbers):
    # This list contains only positive numbers
    max_number = 0
    for n in numbers:
        if n > max_number:
            max_number = n
    return max_number

# Case 1: Works fine
print(f"Max of [1, 5, 2] is: {find_max([1, 5, 2])}")

# Case 2: Fails
print(f"Max of [-10, -5, -2] is: {find_max([-10, -5, -2])}")
  2. Objective: A Rectangle class should calculate its area and perimeter.
    Problem: The get_area method is calculating the wrong value. 
    class Rectangle:
    def __init__(self, length, width):
        self.length = length
        self.width = width

    def get_area(self):
        # Area should be length * width
        return self.length + self.width

    def get_perimeter(self):
        # Perimeter should be 2 * (length + width)
        return (self.length * 2) + (self.width * 2)

# A rectangle of 5x4 has area 20 and perimeter 18
rect = Rectangle(5, 4)

print(f"Area: {rect.get_area()}")         # Expected: 20
print(f"Perimeter: {rect.get_perimeter()}") # Expected: 18
